# 中西研究室ダッシュボード

研究室専用の統合ダッシュボードシステムです。PDF処理、論文翻訳、RAG ChatBotなどの研究効率化ツールを提供します。

## 🚀 主な機能

### 📄 PDF処理
- **高精度テキスト抽出**: 埋め込みテキストとOCRに対応
- **セクション抽出**: 論文の特定章節を自動抽出
- **テキスト整形**: 読みやすい形式に自動整形
- **一括処理**: 複数ファイルの同時処理

### 🌐 論文翻訳・要約
- **GPT-4による高品質翻訳**: 英語→日本語をはじめ多言語対応
- **自動要約生成**: カスタマイズ可能なプロンプト
- **段階的処理**: 長文論文の効率的な翻訳
- **結果の一括ダウンロード**: Markdown形式での出力

### 🤖 RAG ChatBot
- **知識ベース統合**: 過去の卒論・修論を学習
- **セマンティック検索**: 意味理解による高精度回答
- **資料アップロード**: PDFファイルの動的学習
- **引用機能**: 回答の根拠となる資料を表示

### 💎 モダンなUI/UX
- **ダークテーマ**: 目に優しい研究環境
- **レスポンシブデザイン**: デスクトップ・タブレット対応
- **グラスモーフィズム**: 洗練されたビジュアル
- **アニメーション**: スムーズなユーザー体験

## 🛠️ 技術スタック

- **フロントエンド**: Next.js 15, React 19, TypeScript
- **スタイリング**: Tailwind CSS, Lucide Icons
- **バックエンド**: Next.js API Routes
- **AI**: OpenAI GPT-4o, Embeddings API
- **PDF処理**: pdf-parse
- **ファイル処理**: formidable

## 📦 セットアップ

### 必要な環境
- Node.js 18以上
- npm または yarn
- OpenAI APIキー

### インストール手順

1. **依存関係のインストール**
   ```bash
   npm install
   ```

2. **環境変数の設定**
   ```bash
   cp .env.example .env.local
   ```
   
   `.env.local` ファイルに以下を設定：
   ```env
   OPENAI_API_KEY=your_openai_api_key_here
   ```

3. **開発サーバーの起動**
   ```bash
   npm run dev
   ```

4. **ブラウザで確認**
   http://localhost:3000 にアクセス

## 📚 使用方法

### PDF処理
1. 「PDF処理」タブを選択
2. PDFファイルをドラッグ&ドロップまたはファイル選択
3. 必要に応じて抽出設定を調整
4. 処理完了後、結果をダウンロード

### 論文翻訳
1. 「論文翻訳」タブを選択
2. PDFアップロードまたは直接テキスト入力
3. 翻訳設定（言語、モデル、要約設定）を調整
4. 翻訳・要約結果をダウンロード

### RAG ChatBot
1. 「RAG ChatBot」タブを選択
2. 必要に応じて研究資料をアップロード
3. 質問を入力して送信
4. AI回答と参考資料を確認

## 🔧 カスタマイズ

### 翻訳プロンプトの調整
`src/app/api/translate/route.ts` で翻訳の指示を変更可能

### 知識ベースの拡張
`src/app/api/knowledge-base/` でRAGシステムをカスタマイズ可能

### UIテーマの変更
`src/app/globals.css` でカラーパレットやアニメーションを調整
